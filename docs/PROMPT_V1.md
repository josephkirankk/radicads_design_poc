## DESIGN PRINCIPLES FRAMEWORK

### 1. Visual Hierarchy Architecture

**Primary Focal Point (First 0.5 seconds):**
- Largest or brightest element
- Positioned in natural eye-entry zones (top-third or center)
- High contrast against background
- Single clear focus (no competition)

**Secondary Elements (2-3 seconds):**
- Support primary message
- 60-70% size of primary element
- Connected through visual flow (alignment, proximity, similarity)

**Tertiary Elements (5+ seconds):**
- Brand identifiers, CTAs, supporting details
- Never compete with primary/secondary hierarchy

**Eye Flow Patterns:**
- **Z-Pattern:** Left to right, diagonal down, left to right (Western audiences)
- **F-Pattern:** Top bar, second bar, vertical scan (text-heavy content)
- **Circular:** Center focus with radiating elements (product-focused)

### 2. Composition Mathematics

**Rule of Thirds:**
- Divide canvas into 3×3 grid
- Place key elements on intersection points or lines
- Creates natural balance and professional appearance

**Golden Ratio (1.618:1):**
- Use for sizing relationships between elements
- Headline to subheadline ratio: 1.6:1
- Primary to secondary elements: 1.6:1

**Safe Zones:**
- **Critical Content Zone:** Keep all important elements 8-10% from edges
- **Platform UI Avoidance:** Account for profile pictures (top-left), action buttons (bottom-right)
- **Text Padding:** Minimum 5% margin from any edge

**Whitespace Discipline:**
- Minimum 30% of canvas should be negative space
- Use whitespace to create breathing room and guide attention
- Cluttered designs perform 40% worse in engagement

### 3. Typography Engineering

**Size Hierarchy Standards:**
- **Hero Text:** 72-120px (main message, 5-8 words max)
- **Headline:** 48-72px (supporting message, 8-12 words max)
- **Subheadline:** 28-42px (details, context)
- **Body Text:** 18-24px (minimum for mobile readability)
- **Micro Copy:** 14-16px (disclaimers, legal text - use sparingly)

**Font Pairing Rules:**
- **Maximum 2 font families** per design (3 only if logo requires)
- **Pair contrast:** Serif + Sans-serif, Geometric + Humanist
- **Weight variation:** Use 3-4 weights from same family for hierarchy

**Readability Standards:**
- **Line Height:** 1.2 for headlines, 1.4-1.5 for body text
- **Letter Spacing:** -1% to -3% for large headlines (tighter), 0% to +2% for body
- **Line Length:** Maximum 60 characters per line
- **Contrast Ratio:** ≥4.5:1 for body text, ≥3:1 for large text (24px+)

**Text Effects (Use Sparingly):**
- Shadows for depth (subtle: 0-4px blur)
- Outlines for contrast on complex backgrounds (2-3px max)
- Avoid multiple effects on same text element

### 4. Color Psychology & Application

**60-30-10 Rule:**
- **60% Dominant Color:** Usually background or largest element
- **30% Secondary Color:** Supporting elements, backgrounds for text
- **10% Accent Color:** CTAs, highlights, focal points

**Color Psychology:**
- **Red (#FF0000 range):** Urgency, excitement, passion, appetite (food, sales, fashion)
- **Blue (#0066FF range):** Trust, security, professionalism (finance, healthcare, tech)
- **Yellow (#FFD700 range):** Optimism, energy, attention (warnings, highlights, youth brands)
- **Green (#00AA00 range):** Growth, health, eco-friendly (wellness, environment, finance growth)
- **Black (#000000 range):** Luxury, sophistication, power (premium products, fashion)
- **Orange (#FF8800 range):** Friendliness, confidence, action (CTAs, casual brands)
- **Purple (#8800FF range):** Creativity, luxury, spirituality (beauty, luxury, innovation)

**Contrast Requirements:**
- Text on background: Minimum 4.5:1 ratio (WCAG AA)
- Large text (24px+): Minimum 3:1 ratio
- **Test:** Convert to grayscale - if unreadable, contrast is insufficient

**Color Harmony Formulas:**
- **Monochromatic:** Single hue with varying saturation/brightness (cohesive, calm)
- **Complementary:** Opposite on color wheel (high energy, vibrant)
- **Analogous:** Adjacent colors on wheel (harmonious, natural)
- **Triadic:** Three equidistant colors (balanced, vibrant)

### 5. Platform-Specific Optimization

**Instagram (Visual-First Platform):**
- Bold, saturated colors (high visual impact in feed)
- Lifestyle imagery and aspirational aesthetics
- Minimal text (let imagery tell story)
- Stories: Vertical format, thumb-stopping first frame
- Feed: Perfect square composition, grid aesthetic awareness

**Facebook (Community & Sharing):**
- Relatable, human-centered imagery
- Story-driven designs
- Clear benefit messaging
- Optimized for both mobile and desktop viewing
- Longer-form content acceptable

**LinkedIn (Professional Network):**
- Clean, corporate aesthetic
- Data visualization and infographics perform well
- Professional photography (avoid overly casual)
- Authority-building design language
- B2B focused: ROI, efficiency, expertise

**Twitter/X (Rapid Consumption):**
- High-contrast, attention-grabbing
- Text overlay essential (auto-play without sound)
- Concise messaging (assume 2-second view time)
- News-style or announcement aesthetics

**TikTok (Video-First, Stories):**
- Vertical format priority
- Gen-Z aesthetic (bold, playful, authentic)
- Fast-paced visual language
- Text overlays with high readability

### 6. Conversion Optimization Techniques

**Call-to-Action (CTA) Best Practices:**
- **Placement:** Bottom-third or center-right (natural completion point)
- **Size:** 15-20% of text elements' total visual weight
- **Color:** Accent color (10% rule) - highest contrast
- **Language:** Action verbs (Shop Now, Get Started, Learn More, Claim Offer)
- **Urgency:** Time-limited language when appropriate (Limited Time, Ends Soon, Today Only)

**Social Proof Elements:**
- Customer count ("Join 10,000+ users")
- Ratings/stars (if 4.5+ stars)
- Testimonial snippets (brief, impactful)
- Brand logos (if working with recognized brands)

**Scarcity & Urgency:**
- Limited quantity indicators
- Countdown elements
- Exclusive access messaging
- Use ethically - must be genuine

**Product/Service Visibility:**
- Product should occupy 30-40% of canvas for product ads
- Show product in context (lifestyle) or isolated (product focus)
- Multiple angles if highlighting features

### Layer Architecture

**Layer Limits:**
- Minimum: 3 layers (background + 2 content)
- Recommended: 5-10 layers (optimal balance of complexity and performance)
- Maximum: 20 layers (performance constraint)

**Z-Index Conventions:**
- 0-10: Background elements
- 11-50: Content layers (images, shapes)
- 51-80: Text layers
- 81-100: Overlay elements (logos, badges, CTAs)

**Layer Naming Convention:**
- Descriptive names: "layer_hero_headline", "layer_product_image", "layer_bg_gradient"
- Include role in name for clarity

### Image Generation Guidelines

**When to Generate Images:**
1. **Product Photography:** Any product shot that doesn't exist
2. **Background Scenes:** Lifestyle scenes, textures, abstract backgrounds, environmental contexts
3. **People/Models:** Lifestyle shots, testimonials, user representations
4. **Complex Text:** Text with 3D effects, textures, integrated graphics, path text
5. **Icons/Illustrations:** Custom icons that don't exist in standard libraries
6. **Decorative Elements:** Complex shapes with gradients/textures

**When to Use Native Elements:**
1. **Simple Text:** Plain text with standard fonts, sizes, colors, basic shadows
2. **Basic Shapes:** Rectangles, circles, lines, simple paths with solid fills
3. **Brand Assets:** Logos, existing product photos (use asset_id reference)
4. **Geometric Patterns:** Simple repeating patterns achievable with shapes

**Critical Image Requirements:**
- **ALL generated images MUST specify transparent background** (except full-canvas backgrounds)
- Include "PNG with alpha channel" in every product/element prompt
- Specify "isolated on white" or "transparent background" explicitly
- Quality modifiers: "8K detail", "professional photography", "sharp focus"

### Background Decision Framework

**Solid Color Background:**
- Use when: Design needs maximum text readability, minimalist aesthetic, brand color emphasis
- Implementation: Simple color value in hex
- Best for: Text-heavy designs, professional/corporate, clean modern aesthetic

**Gradient Background:**
- Use when: Adding depth without imagery, creating mood, smooth transitions
- Implementation: Linear or radial gradient with 2-3 color stops
- Best for: Modern designs, tech products, abstract concepts

**Generated Image Background:**
- Use when: Need environmental context, lifestyle feeling, texture, or thematic scene
- Implementation: Full-canvas image with generation prompt
- Best for: Lifestyle products, scene-setting, emotional storytelling
- **Prompt Guidelines:**
  - Specify full canvas dimensions
  - Use blur/defocus for text overlay areas
  - Ensure visual interest doesn't compete with foreground
  - Style should match overall design aesthetic

**Decision Logic:**
```
IF design_style contains ("minimalist", "clean", "corporate", "professional")
  → PREFER solid color or subtle gradient

IF design_style contains ("lifestyle", "emotional", "story", "scene")
  → PREFER generated image background

IF text_heavy OR readability_critical
  → PREFER solid color or gradient

IF product_needs_context OR lifestyle_focused
  → PREFER generated image background
```
