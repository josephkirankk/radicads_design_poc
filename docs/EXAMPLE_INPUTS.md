## How to Use These Test Cases:

1. **Copy any test case above** (the entire text in the code block)
2. **Paste it as input** to the prompt system
3. **Receive canonical JSON output** with all design specifications
4. **Validate the output** for:
   - Proper canvas dimensions
   - Image generation prompts with transparent backgrounds
   - Brand consistency
   - Visual hierarchy
   - All coordinates within bounds

Each test case represents a different:
- **Industry** (e-commerce, SaaS, food service, fitness, real estate, electronics, non-profit)
- **Platform** (Instagram, LinkedIn, Facebook, Twitter)
- **Goal** (product launch, feature announcement, promotion, sale, fundraising)
- **Audience** (millennials, professionals, foodies, tech enthusiasts, donors)
- **Style** (modern, professional, fun, luxurious, urgent, heartfelt)

This variety will thoroughly test the prompt's ability to:
- Adapt to different brand personalities
- Choose appropriate backgrounds (color vs gradient vs image)
- Create proper visual hierarchies
- Generate suitable image prompts
- Apply platform-specific best practices
